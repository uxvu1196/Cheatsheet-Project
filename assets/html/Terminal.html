pwd

Zeigt den Namen des aktuellen Arbeitsverzeichnisses an.

ls

Listet Informationen über die DATEIEN auf (standardmäßig das aktuelle Verzeichnis).

Optionen (kleine Auswahl, nur die meistgenutzten):
ls -a: Zeigt alle Einträge auch die versteckten
ls -l: verwendet ein langes Listing-Format (untereinander) bsp.: Dateiname, Datum, Größe, Berechtigungen
ls --help: Dies zeigt die komplette Hilfe für das ls-Kommando an

cd

cd: Man wechselt das aktuelle Arbeitsverzeichnis in der Shell
cd (directory folder)assets: Ändert das aktuelle Verzeichnis in den Ordner Assets.
directory folder auch DIR ist der Parameter zum Befehl cd.

touch

Mit dem touch Befehl wird eine leere Datei erstellt in dem aktuellem Arbeitsverzeichnis (Ordner), es sei denn, es wird -c eingegeben.
Bsp.: touch index.html

Options:
-c: erstellt keine neue Datei
-a: ändert nur die Zugriffszeit
-m: nur die Änderungszeit ändern
▲

cp

Kopiert die QUELLE/Datei in das ZIEL oder mehrere QUELLE(N) in das VERZEICHNIS.

Options:
-i: Fragt nach ob die Zieldatei ersetz werden darf wenn bereits eine Datei vorhanden ist mit dem gleichen Namen
-n: Eine vorhandene Datei wird nicht überschreiben
-r: Alle Unterverzeichnisse und Dateien werden kopiert
-u: kopiert nur, wenn die QUELLE-Datei neuer ist als die Zieldatei oder wenn die Zieldatei fehlt

▲

mv

Verschiebt den Ordner oder die Datei in das neue Ziel oder Datei/Ordner wird umbenannt.

Options:
f: Keine Abfrage vor dem Überschreiben.
verschiebt Datei (filename) in den Ordner (targetfolder) - Bsp.: mv -f (ALter Name) (Neuer Name)
-i: Fragt ob überschrieben werden darf
-u: kopiert nur, wenn die QUELLEN-Datei neuer ist als die ZIEL-Datei oder wenn die Zieldatei fehlt

▲

Entfernt/Löscht (entkoppelt) die DATEI(en).
